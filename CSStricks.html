2 Awesome CSS tricks

Tuesday; October 3, 2017

Introduction

CSS stands for Cascading Style Sheets. CSS describes how HTML elements are to be displayed on screen, paper, or in other media. CSS saves a lot of work. It can control the layout of multiple web pages all at once. External stylesheets are stored in CSS files. Below are 2 CSS tricks to make your website awesome:

#1 Text Outline

What web designers generally do if they need outlined text is that doing the task in Photoshop using the stroke tool. But here is another awesome way. You might have heard of text-shadow option which adds shadow to text.  This property accepts a comma-separated list of shadows to be applied to the text. But we'll use this tool to outline our text.

Below is the code:

HTML body

<p class="your_Div">Awesome Text</p>

CSS

.your_Div {text-shadow: 0 0 4px black, 0 0 4px white, 0 0 4px black, 0 0 4px black, 0 0 4px black;color:green; font-size:2em;}

#2 Responsive Images

Now days many people view your website on mobile, not desktop. Hence it is required to be 100% responsive including the images. This simple snippet can turn all images of your website.

Below is the code

HTML body

<img src="yourimage.png" class="your_Div">Awesome Text</p>

CSS

.your_Div {max-width: 100%; height: auto;}
