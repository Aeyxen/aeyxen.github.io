Photoshop & MarvelApp

Monday; October 2, 2017

Introduction

Adobe Photoshop is a raster graphics editor developed and published by Adobe Systems for MacOS and Windows. It is used for many purposes, which even includes UI designing. Using MarvelApp with photoshop will make this thing even better. 

Follow the steps to do it.

#1 Create a new photoshop document of width 1125px and height 2436px (iPhone X dimensions) 

#2 Design a login form using basic shapes and text something like in this pic

#3 Create another document of same dimensions and design a very basic profile page using shapes and clipping mask, something like in this pic

#4 Go To marvelapp.com and register for an account 

#5 Create a new MarvelApp project and select the device as iPhone X 

#6 upload all the images you made in MarvelApp

#7 Hover the login form image and click on view

#8 Select the login button area by dragging mouse over it

#9 Set Hotspot location to after login page

#10 Chose any Screen Transition if you want

#11 Click on play button

#12 Tada you designed your first app
